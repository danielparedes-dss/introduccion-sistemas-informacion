# Dashboard Ejecutivo MIS 2025

## Descripción

Dashboard ejecutivo desarrollado a partir de datos transaccionales de ventas para apoyar la toma de decisiones gerenciales mediante indicadores clave de desempeño (KPIs).

## KPIs

| KPI | Indicador |
|-----|-----------|
| 1 | Volumen de ventas anual |
| 2 | Ticket promedio por transacción |
| 3 | Categorías activas |
| 4 | Concentración Pareto (80/20) |
| 5 | Producto líder por unidades vendidas |
| 6 | Mes pico de ventas |
| 7 | Tasa de retorno de clientes |
| 8 | Rotación promedio de inventario |

## Tecnologías Utilizadas

- Python
- Google Colab
- Pandas
- Matplotlib
- Plotly
